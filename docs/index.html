Звіт до Лабораторної роботи №3

Тема: Markdown та публікація на GitHub.

1. Короткий опис

У цій лабораторній роботі я досліджую можливості мови розмітки Markdown для створення структурованої технічної документації. Основна мета — навчитися інтегрувати математичні формули, динамічні діаграми та таблиці в один документ. Також в роботі розглядається процес конвертації Markdown-файлів у формати PDF, HTML та DOCX за допомогою утиліти Pandoc.
